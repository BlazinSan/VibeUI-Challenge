# Dumpling Desk | Vibe UI Challenge 2026

Frontend-only frozen dumpling ordering system for the Vibe UI Challenge 2026.

## What It Includes

- Student-facing ordering flow with featured packs, discounts, cart, delivery address, and payment confirmation.
- Frozen dumpling pack menu with search, filters, stable cards, and quick add actions.
- Fast reorder and quick-pick patterns for repeat orders.
- Student reseller application page with required form fields and clear value proposition.
- Track order page with simulated shipment stages and recent order lookup.
- Desktop admin gateway with revenue, shipment status, popular packs, orders, and customer records.
- Static JSON data for 50 products, 300 orders, 250 customers, 250 delivery records, and 300 ratings.
- Graphify integration for Claude Code in `.claude/`.

## Run Locally

```bash
npm install
npm run dev
```

Build check:

```bash
npm run build
```
